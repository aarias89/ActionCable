# README

A mock Slack  Ruby on Rails application showing realtime chat using ActionCable.

## Run Locally

```bash
bundle install
```

```
rails db:create
```

```
rails db:migrate
```

```
rails s -p 30001
```

On browser go to http://localhost:3001/

Create Username and login
 
Open incognito browser go to http://localhost:3001/

Create Username and login
